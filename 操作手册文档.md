# 关系分析器 操作手册

## 一、启动与界面概览

### 1.1 启动应用

- **macOS**：双击 `relationship.app` 或在终端运行可执行文件
- **Windows**：双击 `relationship.exe` 或通过开始菜单启动

### 1.2 主界面布局

主界面从左到右依次为：

- **左侧栏（Sidebar）**：实体类型列表、关系类型列表、按类型筛选、添加新类型
- **中央画布**：图的可视化展示区域
- **右侧属性面板**：选中实体/关系后显示详细信息，可编辑
- **顶部工具栏**：添加实体、添加关系、路径分析、历史记录等快捷入口
- **底部状态栏**：显示 NeuroDB 连接状态、选中信息等

## 二、数据管理操作

### 2.1 添加实体

**方式一：菜单**

- 菜单：**编辑 → 添加实体**（快捷键 `Ctrl/Cmd + N`）
- 或在画布左上角工具栏点击「添加实体」按钮

**方式二：侧边栏**

- 在左侧栏某实体类型旁点击「+」，选择该类型后填写标签和属性

**操作步骤：**

1. 选择实体类型（人物、组织、事件、地点等）
2. 输入标签（必填）
3. 可选添加属性（键值对）
4. 确认后节点会出现在画布上

### 2.2 添加关系

**方式一：菜单**

- 菜单：**编辑 → 添加关系**（快捷键 `Ctrl/Cmd + Shift + N`）
- 或点击工具栏「添加关系」按钮

**操作步骤：**

1. 选择源实体
2. 选择目标实体
3. 选择关系类型（关联、亲属、同事、交易等）
4. 输入关系标签，可选添加属性
5. 勾选「有向」表示方向性

### 2.3 编辑实体/关系

1. 在画布上点击选中实体或关系
2. 右侧属性面板会显示详情
3. 点击「编辑」进入编辑模式
4. 修改标签或属性后点击「保存」

### 2.4 删除实体/关系

- **菜单**：**编辑 → 删除选中**（快捷键 `Backspace`）
- **属性面板**：选中后点击「删除」并确认
- **注意**：删除实体会同时删除与其相关的所有关系

## 三、导入与导出

### 3.1 导入数据

**菜单：文件 → 导入数据...**（快捷键 `Ctrl/Cmd + I`）

支持格式：

| 格式 | 说明 |
|------|------|
| JSON | 完整图数据，含实体、关系、类型定义 |
| CSV 实体 | 表头需包含 id / type / typeId、label / name 及自定义列 |
| CSV 关系 | 表头需包含 source / from、target / to、type / typeId、label |

### 3.2 导出数据

**菜单：文件 → 导出数据...**（快捷键 `Ctrl/Cmd + E`）

- 导出为 JSON 文件，包含全部实体、关系及类型定义
- 可选择保存路径和文件名

### 3.3 加载示例数据

**菜单：文件 → 加载示例数据**

- 自动生成包含人物、组织、事件、地点、电话、账户、车辆、文档及关系的示例图
- 适合快速体验和熟悉功能

### 3.4 清空数据

**菜单：文件 → 清空所有数据**

- 会弹出确认对话框
- 确认后清除所有实体和关系，类型定义保留

## 四、画布与布局

### 4.1 画布操作

| 操作 | 说明 |
|------|------|
| 拖拽画布 | 鼠标按住空白区域拖动 |
| 缩放 | 鼠标滚轮或触控板 |
| 拖拽节点 | 鼠标按住节点拖动以调整位置 |
| 选中节点/边 | 单击实体或关系 |

### 4.2 布局切换

**菜单：布局 → 选择布局**

可用布局：

- 力导向布局（Force Atlas2）
- 同心圆布局
- 环形布局
- 网格布局
- 层次布局（Dagre）
- 辐射布局
- 鱼骨布局
- 生态树布局
- 活动图布局

### 4.3 视图操作

| 菜单 | 快捷键 | 说明 |
|------|--------|------|
| 适配画布 | `Ctrl/Cmd + 0` | 自动缩放以显示全部内容 |
| 放大 | `Ctrl/Cmd + =` | 放大视图 |
| 缩小 | `Ctrl/Cmd + -` | 缩小视图 |

### 4.4 节点折叠与展开

- **折叠**：选中节点后，可将与其直接相连的子节点折叠进该节点
- **展开**：再次选中并展开，恢复子节点显示

## 五、图分析

### 5.1 链接分析（N 度人脉）

**前置条件**：先在画布上选中一个实体节点

**菜单：分析 → 链接分析 (1度/2度/3度)**

- 1 度：直接相连的实体
- 2 度：2 跳内可达的实体
- 3 度：3 跳内可达的实体

**效果**：画布高亮显示分析范围内实体，其他节点会变暗

### 5.2 路径分析

**菜单：分析 → 路径分析...**（快捷键 `Ctrl/Cmd + P`）

或点击工具栏「路径分析」

**操作步骤：**

1. 选择源实体
2. 选择目标实体
3. 执行分析
4. 画布上高亮显示最短路径及其边

### 5.3 群集分析

**菜单：分析 → 群集分析**

- 自动识别图中社区/群组
- 不同群集以不同颜色区分显示

### 5.4 社会网络分析 (SNA)

**菜单：分析 → 社会网络分析 (SNA)**

- 计算度中心性、介数中心性、紧密中心性
- 中心性高的节点在画布上以更大尺寸显示
- 右侧属性面板可查看 Top N 节点排名

### 5.5 清除分析结果

**菜单：分析 → 清除分析结果**（快捷键 `Esc`）

- 清除当前高亮和 SNA 等分析结果

## 十、常见问题

### Q1：数据保存在哪里？

默认保存在：`~/.relationship-analyzer/graph_data.json`（macOS/Linux）或用户主目录下的对应路径。

### Q2：NeuroDB 连接失败怎么办？

- 检查端口 18839 是否被占用
- 内嵌 NeuroDB 需首次下载或解压，网络异常可能导致启动失败
- 即使 NeuroDB 未连接，本地 JSON 仍会正常保存

### Q3：CSV 导入格式有什么要求？

- **实体 CSV**：表头需包含 `type/typeId`、`label/name`，可选 `id` 及自定义列
- **关系 CSV**：表头需包含 `source/from`、`target/to`，可选 `type/typeId`、`label`

### Q4：如何恢复误删数据？

- 若未关闭应用，可通过「历史记录」查看操作
- 建议定期通过「文件 → 导出数据」备份 JSON
